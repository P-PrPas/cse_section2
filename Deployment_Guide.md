# Deployment Guide: EDFVS (Exam Document Face Verification System)

คู่มือฉบับนี้อธิบายถึงขั้นตอนการแปลงโค้ด Python ของโปรเจกต์ EDFVS ให้กลายเป็นแอปพลิเคชันแบบ Standalone (ไฟล์ `.exe`) และการนำแอปพลิเคชันนั้นมาสร้างเป็นตัวติดตั้ง (Setup Wizard) ที่พร้อมส่งมอบให้ลูกค้านำไปติดตั้งในคอมพิวเตอร์ระบบ Windows อย่างง่ายดาย

---

## สถาปัตยกรรมการติดตั้ง
การทำตัวติดตั้งของเราแบ่งออกเป็น 2 ขั้นตอนหลัก:
1. **PyInstaller (`build.bat`)**: ทำหน้าที่แพ็กเกจไฟล์ Python (`.py`), โมเดล AI (`models/`), และไฟล์ UI ทั้งหมด ให้กลายเป็นโฟลเดอร์แอปพลิเคชันเดี่ยว (`dist\EDFVS`) ที่มีไฟล์รัน `EDFVS.exe`
2. **Inno Setup (`setup.iss`)**: ทำหน้าที่นำโฟลเดอร์แอปพลิเคชันเหล่านั้นมาเป็นแผงรวมไฟล์ (Installer) อย่างมืออาชีพ เช่น `EDFVS_Setup_v1.0.exe` พร้อมตั้งค่าหน้าจอและกำหนด Shortcut อัตโนมัติ

---

## 🛠️ ขั้นตอนที่ 1: การ Compile โค้ดเป็นไฟล์ `.exe`
ก่อนอื่น คุณต้องแน่ใจว่าโหลดโมเดลทั้งหมดลงในโฟลเดอร์ `models/` เรียบร้อยแล้ว (เช่น YuNet และ SFace) เพื่อให้ตัวสคริปต์คอมไพล์ทำงานได้สมบูรณ์

1. เปิด Command Prompt (CMD), PowerShell หรือ Terminal ในโปรแกรมเขียนโค้ดของคุณ (เช่น VS Code) 
2. พิมพ์คำสั่งสำหรับสั่งรันไฟล์แบทช์:
   ```cmd
   build.bat
   ```
3. รอจนกว่าระบบจะดำเนินการเสร็จสิ้น (อาจใช้เวลาประมาณ 1-3 นาทีขี้นอยู่กับความเร็วคอมพิวเตอร์)
   - สคริปต์นี้จะทำการติดตั้ง `pyinstaller` (หากยังไม่มี)
   - สคริปต์จำทำการคัดลอกไฟล์ที่เกี่ยวข้องอย่าง `ui/`, `models/` และ `config.json` ให้โดยอัตโนมัติ
4. **ความสำเร็จ:** คุณจะได้โฟลเดอร์ใหม่ชื่อว่า `dist\EDFVS` ซึงสามารถส่งโฟลเดอร์นี้ให้ลูกค้านำไปรัน `EDFVS.exe` แบบไม่ต้องลงโปรแกรมงำให้ยุ่งยากได้เลย

---

## 📦 ขั้นตอนที่ 2: สร้างตัวติดตั้ง Setup แบบพรีเมียม (Installer Wizard)
เพื่อให้ผู้ใช้งานรู้สึกได้ถึงความเป็นมืออาชีพ (ไม่ต้อง Copy โฟลเดอร์วางเอง แต่ใช้วิธีคลิก Next > Next เพื่อเริ่มใช้งาน)

1. ดาวน์โหลดโปรแกรม **Inno Setup Compiler** (เวอร์ชันล่าสุด) ได้[ที่นี่ (เว็บไซต์ทางการ)](https://jrsoftware.org/isdl.php) ฟรี!
2. ติดตั้งโปรแกรม Inno Setup ให้เรียบร้อย
3. กลับไปที่โฟลเดอร์โปรเจกต์ของคุณ แล้ว**ทําการดับเบิลคลิกไฟล์ `setup.iss`** (ระบบจะเปิดไฟล์นี้ด้วยโปรแกรม Inno Setup ขึ้นมา)
4. ในโปรแกรม Inno Setup ด้านบนซ้าย จะมีปุ่มรูป **"ลูกศรสีเขียว"** 🟢 (หรือคลิกเมนู `Build` > `Compile` / กดคีย์บอร์ด `Ctrl+F9`)
5. ทันทีที่ระบบรวมไฟล์เสร็จ ระบบจะสร้างตัวติดตั้งออกมา 
6. **ความสำเร็จ:** ไปที่โฟลเดอร์ `Output/` ในโปรเจกต์ของคุณ คุณจะพบกับไฟล์ตัวติดตั้งที่ชื่อว่า `EDFVS_Setup_v1.0.exe` 

> 💡 **นำไฟล์ `EDFVS_Setup_v1.0.exe` นี้ ไปส่งให้ลูกค้าของคุณได้เลย!** ลูกค้าไม่ต้องลง Python ไม่ต้องเซ็ต Path เพียงแค่คลิกติดตั้ง และเปิดใช้งานแอปพลิเคชันจากหน้า Desktop ตามปกติ
